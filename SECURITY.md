# Security Policy

## Overview

AIRROBOT PRO++ is under active development, and the implementation source code is maintained privately.

The public repository contains documentation and architectural information only.

---

## Reporting Security Issues

If you discover a potential vulnerability or security concern related to the project, please report it responsibly.

Please avoid public disclosure until the issue has been reviewed.

## Scope

Security reports may include:

- Communication vulnerabilities
- Authentication issues
- Backend security concerns
- Robotics networking risks
- API-related vulnerabilities

Responsible disclosure is appreciated.
